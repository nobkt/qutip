# スピンS=1ゼーマン効果シミュレーション実装完了報告

## 概要

スピンS=1系におけるゼーマン効果の量子ダイナミクスを、8つの異なる方法でシミュレーションする包括的なJupyter notebookを作成しました。

## 実装した内容

### 1. メインノートブック: `zeeman_effect_comprehensive.ipynb`

以下の8つのシミュレーション方法を完全に実装：

1. **厳密解 (Exact Solution)**: QuTiPの`mesolve`を使用した行列指数関数による厳密な時間発展計算
2. **鈴木トロッター分解**: 解析的なトロッター分解を用いた時間発展の近似計算
3. **Qiskit Qubit Statevector**: 2量子ビット符号化とQiskitのstatevectorシミュレータ
4. **Qiskit Qubit Shot (ノイズ無)**: ショットベースシミュレーション（統計的サンプリング）
5. **Qiskit Qubit Shot (ノイズ有)**: デポラライジングノイズと振幅減衰を含むショットシミュレーション
6. **MQT Qudit Statevector**: MQT Quditsライブラリによる直接的なqutrit (d=3) statevectorシミュレーション
7. **MQT Qudit Shot (ノイズ無)**: MQTショットベースシミュレーション
8. **MQT Qudit Shot (ノイズ有)**: ノイズを含むMQTショットシミュレーション

## 技術的詳細

### 物理系

**ハミルトニアン**:
```
H = -ω_L * J_z
```

ここで、`ω_L = 2π × 1.0 rad/s`（ラーモア周波数）

**初期状態**: `|ψ(0)⟩ = (|1,+1⟩ + |1,0⟩)/√2`

### Qubitエンコーディング

3準位系を2量子ビットに符号化：
- `|m=+1⟩ → |00⟩`
- `|m=0⟩ → |01⟩`
- `|m=-1⟩ → |10⟩`
- `|11⟩` は未使用

### Quditアプローチ

MQT Quditsを使用した直接的な3準位（qutrit）表現：
- ネイティブなqutritゲート
- より効率的な回路表現
- Qubitエンコーディングの必要なし

## 厳密性の保証

### ヒューリスティック処理なし

以下の点で厳密性を保証：

1. **厳密な理論に基づく実装**
   - 行列指数関数の数値計算（QuTiP）
   - 鈴木トロッター分解の理論的に正当化された近似
   - ユニタリ演算子の正確な実装

2. **Fallbackメカニズムなし**
   - すべての計算が失敗する可能性を考慮
   - エラーハンドリングは明示的
   - 近似を使用する場合は明確に文書化

3. **検証可能な実装**
   - すべてのコードが読みやすく理解しやすい
   - 中間結果を確認可能
   - 理論と実装の対応が明確

### 精度の評価

各方法の精度を以下の指標で評価：

1. **最大誤差**: `max|P_method(t) - P_exact(t)|`
2. **RMS誤差**: `sqrt(mean((P_method - P_exact)²))`
3. **統計誤差**: ショットシミュレーションの理論的誤差
