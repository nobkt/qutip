#### 1.2 スピン角運動量演算子
- S=1のスピン演算子の3×3行列表現を明示：
  ```
  Jx = (1/√2) [0  1  0]    Jy = (1/√2) [0 -i  0]    Jz = [1  0  0]
  ```

#### 1.3 ゼーマンハミルトニアン

#### 1.4 固有状態とエネルギー固有値
- 固有状態 |m⟩ (m = +1, 0, -1) の定義
- エネルギー固有値 Em = -ωL m の導出
- エネルギースペクトル:
  - E₊₁ = -ωL (最低エネルギー)
  - E₀ = 0 (中間エネルギー)
  - E₋₁ = +ωL (最高エネルギー)

#### 1.5 時間発展演算子
- シュレーディンガー方程式からの形式解の導出
- U(t) = exp(-iHt/ℏ) = exp(iωLt Jz)
- 対角行列の指数関数の明示的な形:
  ```
  U(t) = [exp(iωLt)    0           0        ]
         [0             1           0        ]
         [0             0         exp(-iωLt) ]
  ```

#### 1.6 期待値の時間発展
- ハイゼンベルグ方程式 d⟨A⟩/dt = (i/ℏ)⟨[H, A]⟩ の適用
- ⟨Jz⟩ が保存量であることの証明
- ⟨Jx⟩ と ⟨Jy⟩ が ωL で歳差運動することの導出:
  ```
  ⟨Jx(t)⟩ = ⟨Jx(0)⟩cos(ωLt) - ⟨Jy(0)⟩sin(ωLt)
  ⟨Jy(t)⟩ = ⟨Jx(0)⟩sin(ωLt) + ⟨Jy(0)⟩cos(ωLt)
  ⟨Jz(t)⟩ = ⟨Jz(0)⟩
  ```

### 2. 各シミュレーション手法の理論的説明

#### 方法1: 厳密解 (Exact Solution)
- **理論**: 行列指数関数の数値的厳密計算
- **実装**: QuTiPの`mesolve`による対角化と指数関数計算
- **誤差**: 浮動小数点丸め誤差のみ (~10⁻¹⁵)
- **物理的予測**: 占有確率が時間不変（固有状態の重ね合わせ）

#### 方法2: 鈴木トロッター分解 (Suzuki-Trotter Decomposition)
- **理論**: トロッター公式 exp(-i(A+B)Δt) ≈ exp(-iAΔt/2)exp(-iBΔt)exp(-iAΔt/2)
- **適用**: H = -ωL Jz は単一項なので形式的には厳密
- **目的**: 実装の枠組みを示し、数値誤差を評価
- **誤差**: 理論的には O(Δt³)、実際には数値誤差のみ

#### 方法3-5: Qiskit Qubit シミュレーション
- **理論**: 3次元ヒルベルト空間を2量子ビット（4次元）にエンコード
- **エンコーディング**:
  - |m=+1⟩ ↔ |00⟩
  - |m=0⟩ ↔ |01⟩
  - |m=-1⟩ ↔ |10⟩
  - |11⟩ は補助空間（未使用）
- **方法3**: Statevectorシミュレータ（確率振幅を直接計算）
- **方法4**: Shotシミュレータ（ノイズ無、統計誤差 ~1/√Nshots）
- **方法5**: Shotシミュレータ（depolarizing + amplitude dampingノイズ）

#### 方法6-8: MQT Qudit シミュレーション
- **理論**: qutrit（d=3）を直接扱う
- **利点**: エンコーディング不要、補助量子ビット不要
- **方法6**: Qutrit statevectorシミュレータ
- **方法7**: Qutrit shotシミュレータ（ノイズ無）
- **方法8**: Qutrit shotシミュレータ（qutrit固有のノイズ）
