## 物理パラメータ

### エネルギー準位
- E_S0 = 0.0 eV（基底一重項状態）
- E_T1 = 1.5 eV（励起三重項状態）
- E_S1 = 2.2 eV（励起一重項状態）

### 相互作用
- V_ET = 0.05 eV（エネルギー移動結合定数）
- V_TTA = 0.03 eV（三重項-三重項消滅結合定数）

### シミュレーションパラメータ
- 時間範囲: 0 - 100 ps
- トロッターステップ: 0.5 ps
- トロッター次数: 2
- ショット数: 10,000

## 状態空間

### 物理的状態（5状態）
1. |T₁S₀⟩: 分子AがT₁、分子BがS₀
2. |S₀T₁⟩: 分子AがS₀、分子BがT₁
3. |T₁T₁⟩: 両分子がT₁
4. |S₁S₀⟩: 分子AがS₁、分子BがS₀
5. |S₀S₁⟩: 分子AがS₀、分子BがS₁

### 表現の次元
- **古典モデル**: 5次元ベクトル
- **量子厳密解**: 5次元複素ベクトル
- **Qubit表現**: 16次元（4量子ビット）、うち5状態を使用
- **Qudit表現**: 9次元（2 qutrit）、うち5状態を使用

## 比較結果

### リソース効率
- **Qubit**: 4量子ビット（16次元空間）
- **Qudit**: 2 qutrit（9次元空間）
- **効率比**: Quditの方が約1.78倍コンパクト

### 計算精度
- **厳密解**: 基準値
- **トロッター2次**: 十分な精度（最大誤差 < 10⁻⁴）
- **古典モデル**: 定性的には一致、定量的には差異あり

### 実装アルゴリズム

#### 鈴木-トロッター分解（2次）
```
U(Δt) ≈ exp(-iH₀Δt/2) exp(-iH_ETΔt/2) exp(-iH_TTAΔt) exp(-iH_ETΔt/2) exp(-iH₀Δt/2)
```

#### 時間発展
```
|ψ(t)⟩ = [U(Δt)]^(t/Δt) |ψ(0)⟩
```

#### ポピュレーション計算
```
p_i(t) = |⟨i|ψ(t)⟩|²
```
