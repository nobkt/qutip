- 直交性：$\text{Tr}(W_{k,l}^\dagger W_{k',l'}) = 3\delta_{k,k'}\delta_{l,l'}$
- 完全性：任意の $3 \times 3$ 行列の展開が可能

#### 6.1.4 物理的演算子の構成

**射影演算子**:
```
|0⟩⟨0| = (1/3)(I + Z + Z²)
|1⟩⟨1| = (1/3)(I + ω²Z + ωZ²)
|2⟩⟨2| = (1/3)(I + ωZ + ω²Z²)
```

すべての射影演算子について：
- 完全性: $P_0 + P_1 + P_2 = I$
- 直交性: $P_i P_j = \delta_{ij} P_i$
- 冪等性: $P_i^2 = P_i$

を明示的に検証しました。

**遷移演算子**:
```
|1⟩⟨0| = X|0⟩⟨0| = (1/3)(X + XZ + XZ²)
|0⟩⟨1| = (|1⟩⟨0|)† = (1/3)(X² + Z²X² + ZX²)
```

エルミート遷移演算子:

すべての計算を完全に展開して示しました。
