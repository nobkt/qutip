The shot-based simulation sections were:
1. Computing exact time evolution: U = exp(-iHt)
2. Directly applying: psi_t = U * psi0
3. Just initializing the final state: qc.initialize(psi_t_qubit)
4. Measuring without any evolution gates

This is not a proper quantum simulation!

SOLUTION IMPLEMENTED:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GATE DECOMPOSITION
   • Decompose single time step using KAK decomposition
   • Convert U_step into RX, RY, RZ, CX gates
   • Use TwoQubitBasisDecomposer from Qiskit

2. CUMULATIVE CIRCUIT BUILDING
   • Start from initial state psi0
   • Loop through time steps: for step in range(t_idx)
   • Compose gate sequence: qc.compose(qc_step)
   • Build complete evolution through gates

3. QUBIT ORDERING FIX
   • Handle QuTiP (big-endian) ↔ Qiskit (little-endian)
   • Apply permutation [0, 2, 1, 3] to states and operators
