Basic Usage:
-----------
from qudit.qudit import convert_hamiltonian_to_mqt_circuit
import numpy as np

# Define Hamiltonian
ops = get_spin1_operators()
H = -2*np.pi * ops['Jz']

# Convert to MQT circuit
circuit, info = convert_hamiltonian_to_mqt_circuit(
    H, time=1.0, trotter_steps=10, trotter_order=2
)

# Access results
print(circuit.to_qasm())               # DITQASM output
print(f"Gates: {info['num_gates']}")   # Gate count
print(info['gates'][0]['unitary'])     # First gate matrix

================================================================================
MATHEMATICAL FOUNDATION
================================================================================

Spin-1 Operators (ℏ=1):
-----------------------
Jz = diag(1, 0, -1)

Jx = (1/√2) [0  1  0]
            [1  0  1]
            [0  1  0]

Jy = (1/√2) [0 -i  0]
            [i  0 -i]
            [0  i  0]

Evolution Operators:
-------------------
Rx(θ) = exp(-iθJx)
Ry(θ) = exp(-iθJy)
Rz(θ) = exp(-iθJz)

Computed exactly: U = scipy.linalg.expm(-1j * angle * operator)

Trotter Decomposition:
---------------------
Order 1: U(Δt) ≈ exp(-iH₁Δt)·exp(-iH₂Δt)·exp(-iH₃Δt)
         Error: O(Δt²)

Order 2: U(Δt) ≈ exp(-iH₁Δt/2)·...·exp(-iHₙΔt/2)·...·exp(-iH₁Δt/2)
         Error: O(Δt³)

Order 4: Uses Suzuki's fractal composition
         Error: O(Δt⁵)
