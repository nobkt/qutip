## 技術的詳細

### 物理パラメータ

**エネルギー準位**:
- E_S0 = 0.0 eV（基底一重項状態）
- E_T1 = 1.5 eV（励起三重項状態）
- E_S1 = 2.2 eV（励起一重項状態）

**相互作用パラメータ**:
- V_ET = 0.05 eV（エネルギー移動結合定数）
- V_TTA = 0.03 eV（TTA結合定数）

**シミュレーションパラメータ**:
- 時間範囲: 0 - 100 ps
- 時間ステップ数: 200
- トロッターステップサイズ: 0.5 ps
- トロッター次数: 2（2次Strang分割）
- ショット数: 10,000

### 状態空間

**物理的状態**（5状態）:
1. |T₁S₀⟩: 初期状態（分子AがT₁、分子BがS₀）
2. |S₀T₁⟩: エネルギー移動後
3. |T₁T₁⟩: 両分子励起状態
4. |S₁S₀⟩: TTA生成物（分子AがS₁）
5. |S₀S₁⟩: TTA生成物（分子BがS₁）

**表現の次元**:
- 古典モデル: 5次元実数ベクトル
- 量子厳密解: 5次元複素ベクトル
- Qubit表現: 16次元（4量子ビット、5状態使用）
- Qudit表現: 9次元（2 qutrit、5状態使用）

### 実装アルゴリズム

**鈴木-トロッター分解（2次）**:
```
U(Δt) ≈ exp(-iH₀Δt/2) exp(-iH_ETΔt/2) exp(-iH_TTAΔt) 
        exp(-iH_ETΔt/2) exp(-iH₀Δt/2)
```

**時間発展**:
```
|ψ(t)⟩ = [U(Δt)]^(t/Δt) |ψ(0)⟩
```

**ポピュレーション計算**:
```
p_i(t) = |⟨i|ψ(t)⟩|²
```
